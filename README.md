# GrayCat QR Tool

Simple and fast QR code generator with PNG and SVG export.  
Works entirely in the browser. No limits, no tracking.

---

## 🌐 Live version

https://mikkrak.github.io/qr-tool/

---

## ✨ Features

- Generate QR codes from:
  - URLs
  - Text
- Instant preview
- Download as:
  - PNG
  - SVG
- Works without server
- Clean and minimal UI

---

## 🖥 Usage

1. Open the site  
2. Choose mode:
   - Link
   - Text  
3. Enter data  
4. Click **Generate**  
5. Download QR (PNG or SVG)
